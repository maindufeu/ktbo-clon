# ktbo-bi
Adverity scripting

Aquí irán los scripts que ayudan a tener una correcta ingesta de datos en adverity para disminuir el número de iteraciones.

Son necesarios principalmente para archivos manuales como:
- Planes de medios
- Listenings
- Archivos de campaña
